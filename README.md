
## Executar

```sh
yarn start
yarn android
yarn ios
```

## Recarregar

```sh
# android
rr
cmd + m

# ios
cmd + r
cmd + d

```

## Adicionar o typescript

- https://reactnative.dev/docs/typescript

```sh
yarn add -D typescript @types/jest @types/react @types/react-native @types/react-test-renderer
```
